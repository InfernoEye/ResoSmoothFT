# SmoothFaceTracking port to RML Original by [Purpzie](https://github.com/Purpzie/ResoSmoothFT)

A [ResoniteModLoader](https://github.com/resonite-modding-group/ResoniteModLoader) mod for [Resonite](https://resonite.com/) that smooths your face tracking locally before it gets synced with other players.

## Manual Installation
1. Install [ResoniteModLoader](https://github.com/resonite-modding-group/ResoniteModLoader).
2. Download the latest release of [SmoothFaceTracking.dll](https://github.com/InfernoEye/ResoSmoothFT/releases/latest/download/SmoothFaceTracking.dll) and place it into your `rml_mods` folder. This folder should be at `C:\Program Files (x86)\Steam\steamapps\common\Resonite\rml_mods` for a default install. You can create it if it's missing, or if you launch the game once with ResoniteModLoader installed it will create the folder for you.
3. Start the game. If you want to verify that the mod is working you can check your Resonite logs.

## Usage
Once installed, your face tracking will be smoothed by default.
Smoothing speed and toggles can be found in your settings under (For that you need [ResoniteModSettings](https://github.com/badhaloninja/ResoniteModSettings/releases/latest/download/ResoniteModSettings.dll
)) `Mods > SmoothFaceTracking`.
